Deletes an item from the board.

### Parameters

| Parameters | Type   | Description                        |
| ---------- | ------ | ------------------------------------ |
| options    | Object | `instanceId` (String, required).     |

### Syntax

```js
await boardInstance.deleteItem({ instanceId: "item_123" });
```
