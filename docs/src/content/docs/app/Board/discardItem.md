Discards a board item.

### Parameters

| Parameters | Type   | Description                        |
| ---------- | ------ | ------------------------------------ |
| options    | Object | `instanceId` (String, required).     |

### Syntax

```js
await boardInstance.discardItem({ instanceId: "item_123" });
```
