Submits/finalizes a dataform item.

### Parameters

| Parameters | Type   | Description                                                                                  |
| ---------- | ------ | -------------------------------------------------------------------------------------------- |
| options    | Object | `itemId` (String, required); `data` (Object, optional); `viewId` (String, optional).          |

### Syntax

```js
await dataformInstance.submitItem({ itemId: "item_123" });
```
