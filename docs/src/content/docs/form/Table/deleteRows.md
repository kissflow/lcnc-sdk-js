Deletes multiple rows from the specified [table instance](/form/gettable/).

### Parameter

| Parameters | type             |
| ---------- | ---------------- |
| rows       | Array of Strings |

### Syntax

```js
let rows = ["rowId1", "rowId2"];
tableInstance.deleteRow(rows);
```
